# Sistema-Farmacia-POS
# 💊 Sistema de Gestión Farmacéutico (POS)

Sistema de escritorio desarrollado en Java para la gestión integral de una farmacia: ventas, inventario, caja, clientes y proveedores.

# Tecnologías

- Java (Swing)
- MySQL
- JDBC
- Arquitectura en capas (DAO / VO / UI)

# Funcionalidades

- Punto de venta (POS) con carrito dinámico
- Control de stock en tiempo real
- Alertas de productos por vencer
- Gestión de clientes y proveedores
- Control de caja (apertura/cierre)
- Historial de ventas con detalle y anulación
- Sistema de roles (Admin / Cajero)

# Características técnicas

- Programación orientada a objetos (POO)
- Separación de capas (DAO Pattern)
- Manejo de eventos (Swing)
- Persistencia con MySQL
